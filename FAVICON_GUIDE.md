### Option 2: Favicon.io
1. Go to: https://favicon.io
2. Choose "Text" or "Image"
3. For Text: Enter "SS" or "S" (your initials)
4. Customize colors and font
5. Download and extract to root directory

### Option 3: Create Manually
Use any image editor (Photoshop, GIMP, Canva):
- Create 512x512px image
- Export as:
  - favicon.ico (16x16, 32x32)
  - favicon-16x16.png
  - favicon-32x32.png
  - apple-touch-icon.png (180x180)
